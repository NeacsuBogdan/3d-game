# Contributing

## Branching
- Trunk-based: \main\ is stable; small feature branches; PR required.

## PR Checklist
- [ ] \
pm run lint\ passes
- [ ] \
pm run typecheck\ passes
- [ ] \
pm run test\ passes
- [ ] \
pm run build\ passes
- [ ] Docs updated if applicable

## Node & Env
- Node 22 LTS (see \.nvmrc\)
- Do not commit \.env.local\. Use \.env.example\ as template.
